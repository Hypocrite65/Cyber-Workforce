# UI Designer Skill

## 功能
基于产品需求自动生成：
- 配色方案（主题色、辅助色、语义色）
- 设计 Tokens（CSS 变量、Tailwind 配置）
- 组件库推荐（Material-UI、Ant Design、shadcn/ui）
- 布局建议（响应式栅格系统）

## 技术栈支持
- **Web**: TailwindCSS, CSS-in-JS, CSS Variables
- **Design Systems**: Material Design 3, Fluent UI, Carbon Design

## 输出格式
- JSON 格式的设计 Token
- 可直接使用的 CSS/SCSS 代码
- Figma 风格的设计规范文档
